# 图标说明

由于无法直接生成 PNG 图标，请按以下步骤创建图标：

## 方法1：使用在线工具生成

1. 访问 https://www.favicon-generator.org/
2. 上传任意图片或使用文字生成
3. 下载 16x16, 48x48, 128x128 三个尺寸
4. 重命名为 icon16.png, icon48.png, icon128.png
5. 放入 icons 文件夹

## 方法2：使用现有图标

暂时可以使用任意 PNG 图片，重命名为对应尺寸即可。

## 方法3：使用 Emoji

可以使用 🎯 或 ✅ 等 emoji 截图作为图标。

---

**临时方案：扩展可以在没有图标的情况下运行，只是显示默认图标。**
